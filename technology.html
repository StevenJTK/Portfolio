HTML

Hyper Text Markup Language (HTML) is the core

foundation of a website. It is essentially the building

blocks of all websites. HTML focuses on describing

the structure of a web page, as well as showing content.

The language does not offer much in terms of design choices.

CSS

Cascading Style Sheets (CSS) solved the issue that HTML could not.

It allowed the development of large websites and has been the standard

use for designing websites since. This current website serves as my

acquired knowledge of both HTML & CSS as a junior developer.
